# Task 4: Data Visualization

## Objective

The objective of this task is to represent data visually using charts and graphs to make trends, patterns, and insights easier to understand.

## Activities Performed

* Visualized top-selling products
* Analyzed payment method preferences
* Examined order status distribution
* Studied monthly revenue trends
* Compared revenue generated through different referral sources

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Outcome

The visualizations helped in understanding customer purchasing behavior, sales performance, and business trends more effectively. Graphical representation made the insights easier to interpret and communicate.
